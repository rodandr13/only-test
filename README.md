# Исторические даты

Демонстрационный проект с использованием React, TypeScript, SCSS, gsap, Swiper.
Адаптированный под мобильный и декстопные размеры экранов. 


**Установить зависимости**

   ```bash
   npm install
   ```

**Сделать билд**

   ```bash
   npm run build
   ```
   
**Запустить статический сервер с билдом**

   ```bash
   npm run start
```
